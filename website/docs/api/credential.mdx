# Credential API

`CredentialHandle` manages n8n credentials — create, read, update, delete, test, and transfer between projects.

## Access

```ts
const credential = client.credential();
```

## Methods

### `list(params?)`

List credentials.

```ts
const { data, nextCursor } = await client.credential().list({
  limit: 10,
  cursor: 'abc123',
});
```

### `get(id)`

Get a credential by ID.

```ts
const credential = await client.credential().get('cred-123');
```

### `create(data)`

Create a new credential.

```ts
const created = await client.credential().create({
  name: 'AWS Credentials',
  type: 'aws',
  data: { accessKey: 'AKIA123', secretKey: 'secret456' }, // pragma: allowlist secret
  projectId: 'proj-1',
});
```

### `update(id, data)`

Update a credential.

```ts
const updated = await client.credential().update('cred-123', {
  name: 'Updated AWS Credentials',
  data: { accessKey: 'AKIA789', secretKey: 'newsecret' }, // pragma: allowlist secret
});
```

### `delete(id)`

Delete a credential.

```ts
const deleted = await client.credential().delete('cred-123');
```

### `test(id)`

Test a credential's connection.

```ts
const result = await client.credential().test('cred-123');
// result.success, result.message
```

### `transfer(id, destinationProjectId)`

Move a credential to another project.

```ts
await client.credential().transfer('cred-123', 'proj-456');
```

### `getSchema(credentialTypeName)`

Get the schema for a credential type.

```ts
const schema = await client.credential().getSchema('httpHeaderAuth');
```
