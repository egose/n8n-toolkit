# Data Table API

`DataTableHandle` manages n8n data tables — tables, columns, and rows with filtering and upsert support.

This is one of the most flexible handles in the client. The row methods have typed overloads so the return type changes based on flags like `returnType` and `returnData`.

## Access

```ts
const dataTable = client.dataTable();
```

## Methods

## Common Tasks

- create tables for lightweight operational data
- insert rows in batches
- update rows with filters
- upsert one logical record based on a filter
- manage table columns separately from row data

### `list(params?)`

List data tables.

```ts
const { data, nextCursor } = await client.dataTable().list({
  limit: 10,
  filter: 'name',
  sortBy: 'createdAt:desc',
});
```

### `get(id)`

Get a data table by ID.

```ts
const table = await client.dataTable().get('dt-123');
```

### `create(data)`

Create a new data table.

```ts
const table = await client.dataTable().create({
  name: 'User Events',
  columns: [
    { name: 'event', type: 'string' },
    { name: 'createdAt', type: 'date' },
  ],
});
```

### `update(id, data)`

Update a data table.

```ts
const updated = await client.dataTable().update('dt-123', {
  name: 'User Events v2',
});
```

### `delete(id)`

Delete a data table.

```ts
await client.dataTable().delete('dt-123');
```

## Row Operations

### `listRows(dataTableId, params?)`

List rows in a data table.

```ts
const { data, nextCursor } = await client.dataTable().listRows('dt-123', {
  filter: 'event=login',
  sortBy: 'timestamp:desc',
  search: 'alice',
});
```

### `insertRows(dataTableId, data)`

Insert rows into a data table.

```ts
const rowIds = await client.dataTable().insertRows('dt-123', {
  data: [
    { event: 'login', createdAt: '2024-01-01T00:00:00Z' },
    { event: 'logout', createdAt: '2024-01-01T01:00:00Z' },
  ],
  returnType: 'id',
});
```

### `updateRows(dataTableId, data)`

Update rows matching a filter.

```ts
const updatedRows = await client.dataTable().updateRows('dt-123', {
  filter: {
    filters: [{ columnName: 'event', condition: 'eq', value: 'login' }],
  },
  data: { event: 'sign_in' },
  returnData: true,
});
```

### `upsertRow(dataTableId, data)`

Insert or update a single row.

```ts
const row = await client.dataTable().upsertRow('dt-123', {
  filter: {
    filters: [{ columnName: 'event', condition: 'eq', value: 'login' }],
  },
  data: { event: 'login', createdAt: '2024-01-01T00:00:00Z' },
  returnData: true,
});
```

### `deleteRows(dataTableId, params)`

Delete rows matching a filter.

```ts
const result = await client.dataTable().deleteRows('dt-123', {
  filter: JSON.stringify({
    type: 'and',
    filters: [{ columnName: 'event', condition: 'eq', value: 'logout' }],
  }),
  returnData: true,
  dryRun: false,
});
```

## Column Operations

### `listColumns(dataTableId)`

List columns in a data table.

```ts
const columns = await client.dataTable().listColumns('dt-123');
```

### `createColumn(dataTableId, data)`

Add a column to a data table.

```ts
const column = await client.dataTable().createColumn('dt-123', {
  name: 'user_id',
  type: 'string',
});
```

### `updateColumn(dataTableId, columnId, data)`

Update a column.

```ts
const updated = await client.dataTable().updateColumn('dt-123', 'col-1', {
  name: 'user_identifier',
});
```

### `deleteColumn(dataTableId, columnId)`

Delete a column.

```ts
await client.dataTable().deleteColumn('dt-123', 'col-1');
```

## Return Type Narrowing

Some row operations change their return type based on the request flags:

```ts
const count = await client.dataTable().insertRows('dt-123', {
  data: [{ event: 'login' }],
});
// { count: number }

const ids = await client.dataTable().insertRows('dt-123', {
  data: [{ event: 'login' }],
  returnType: 'id',
});
// number[]
```
